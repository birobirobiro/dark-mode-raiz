## 📥 Como baixar o projeto

```bash

  # Clonar o repositório
  $ git clone https://github.com/birobirobiro/dark-mode-raiz.git

  # Acessar o diretório
  $ cd dark-mode-raiz

  # Executando o projeto
  $ code .


  # Visual Studio Code
  * Clicar com o botão direito em cima de index.html
  * Abrir com o Live Server
```
